# 必要資料リスト

## すでに手元にある資料

- 会社案内
- 直近の売上推移
- カタログ制作の見積書

## 追加で依頼したい資料

1. 展示会ブース設営の見積書
   - 対象経費{{AMOUNT_002}}の根拠資料として必要です。
   - 設営内容、数量、単価、納期が分かるものを依頼してください。

2. Webページ改修の見積書
   - 対象経費{{AMOUNT_003}}の根拠資料として必要です。
   - 加工事例ページ、問い合わせ導線、掲載原稿作成の範囲を分けて記載してもらうと確認しやすいです。

3. 事業目的の説明文
   - {{CUSTOM_004}}が、展示会後にどの顧客層からどのような問い合わせを増やしたいのかを確認してください。
   - 小ロット対応、試作品対応、短納期対応のうち、優先して訴求したい強みを確認してください。

4. 展示会情報
   - 展示会名、開催予定日、出展予定内容、想定来場者を確認してください。
   - {{DATE_002}}までに確認できる範囲で構いません。
